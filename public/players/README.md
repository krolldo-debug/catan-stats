# Profilfotos

Lege hier die drei Spielerfotos ab — **exakt mit diesen Dateinamen** (Kleinbuchstaben, Endung `.jpg`):

- `dominic.jpg`  → Gym-Mirror-Selfie
- `dante.jpg`    → lächelnd im Wintergarten
- `carl.jpg`     → mit blauer Sonnenbrille

Solange eine Datei fehlt, zeigt die App automatisch einen farbigen Platzhalter-Avatar
(z. B. „Do", „Da", „Ca"). Sobald die Fotos hier liegen und deployed sind, erscheinen sie.

Quadratische Bilder sehen am besten aus, sind aber nicht Pflicht — die App schneidet
automatisch rund zu.
